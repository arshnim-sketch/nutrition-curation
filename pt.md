이번 프로젝트는 AI 기반 서비스 / 시스템을 설계하고 구현하는 것이 목표입니다.
학생들은 LLM 기반 시스템을 직접 설계하고 구현하며
문제 정의부터 아키텍처 설계, 구현, 평가까지 전 과정을 경험하게 됩니다.
### 🔥 프로젝트 목표
다음 3가지를 모두 만족하는 프로젝트를 만드는 것이 목표입니다.
### 1️⃣ 실제 문제 해결
단순 기능 구현이 아니라 명확한 문제 해결이 있어야 합니다.
예시
문서 분석 AI
업무 자동화 에이전트
데이터 분석 AI
개인 생산성 도구
지식 기반 QA 시스템
멀티 에이전트 시스템
### 2️⃣ LLM 시스템 설계

LangChain

### 3️⃣ 설명 가능한 프로젝트
좋은 프로젝트의 기준은
“얼마나 복잡한가”가 아니라“얼마나 명확하게 설명할 수 있는가”
다음 질문에 답할 수 있어야 합니다.

어떤 한계가 있었는가
